# CPT208-A3-2
museum
## 👥 User Persona

| User Group | Core Pain Points | Core Needs |
|------------|-------------------|------------|
| **Child** | Museum exhibits are "look-only" and distant from daily life, leading to a boring experience. | Desire hands-on interaction and playful engagement, avoiding passive, didactic explanations. |
| **Young Adult** | Outdated exhibition design, less efficient information access than web searches, and unengaging display formats. | Seek gamified challenges or modern interactive formats (e.g., comment-style interactions) to spark curiosity. |
